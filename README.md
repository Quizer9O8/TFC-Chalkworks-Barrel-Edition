# TFC Chalk Addon
<img width="540" height="540" alt="Chalk_TFC_Addon" src="https://github.com/user-attachments/assets/4ee9e8ac-25e5-476e-9c8e-331cd1dbca54" />

A datapack that adds recipes to make different colored chalk sticks in TerraFirmaCraft from the mod [Chalk](https://www.curseforge.com/minecraft/mc-mods/chalk).

Inlcudes support for other TFC mod addons that add barrels as well such as ArborFirmaCraft.
